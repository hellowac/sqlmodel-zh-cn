# 连接表格 - JOIN - 简介

到目前为止，你已经知道如何使用 **SQLModel** 在单个表格上执行主要的 <abbr title="Create, read, update, delete.">CRUD</abbr> 操作了。🎉

但 SQL 数据库的主要优势和特性是能够处理关联数据，**连接** 或 **“连接”** 不同的表格。将一个表格中的行与另一个表格中的行连接起来。

接下来，让我们看看如何使用 **SQLModel** 来管理连接的数据。🤝
